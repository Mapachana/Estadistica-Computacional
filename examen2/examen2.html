Ejercicio 2

Comenzamos definiendo las funciones necesarias respectivas a la distribucin de Pareto.

# Funcin de densidad
dpareto <- function(x,a,b){
  return( (a*b^a)/(x^(a+1)) *(x>=b) )
}

# Funcin de distribucin
ppareto <- function(x,a,b){
  return( (1-(b/x)^a) *(x>=b) )
}

# Inversa
invpareto <- function(u,a,b){
  return( b/((1-u)^(1/a)) )
}

n <- 1000
a <- 5
b <- 4

u <- runif(n)
x <- invpareto(u, a, b)

# Evaluacin mediante grficos y Kolmogorov-Smirnov
hist(x, freq=FALSE, breaks="FD", xlim=c(3.5,10), ylim=c(0, 1.2), main="Mtodo de inversin para la distribucin de Pareto")
lines(density(x), col="blue")
curve(dpareto(x,a,b), add=TRUE, col="red")

ks.test(x, ppareto, a=a, b=b)
## 
##  One-sample Kolmogorov-Smirnov test
## 
## data:  x
## D = 0.018977, p-value = 0.8641
## alternative hypothesis: two-sided

Vemos por el grfico y el p-valor de Kolmogorov-Smirnov que en efecto los valores generados provienen de una distribucin de Pareto.
